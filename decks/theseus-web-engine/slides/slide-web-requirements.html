Key Requirements
Cross-Platform Plasticity
Platform feature modularization to allow for quickly adding new operating system support as well as new core capabilities such as WebGPU, Bluetooth, and adjacent platform services.
Build-out of Windows and Linux Support
Filling in Windows and Linux implementations for the initial cross-platform roll-out, primarily across graphics support, event and window-system support, networking support, and integrations support.
Separate-Process GPU Rendering
Full extraction of graphics implementation from the Web Process into a dedicated GPU Process with Display List support in order to support modernized graphics and font-rendering pipelines.
Persistent Data Structures Re-architecture
Re-architecting the key underlying data structures to support persistence in order to enable sharing and saving of tabs.
Browser-in-Browser Implementation
The ability to write major portions of the engine and the browser “in the engine itself” by using web technologies.
BIB Features
General bucket for features that will either be written entirely or partially using BIB, such as many of the collaboration and sharing features.
