Key Requirements
Efficient GPU
Support for modern tile-based deferred rendering vs. traditional immediate mode rendering to allow for an always-on efficient GPU.
Native Hypervisor Support
Cache partitioning, interrupt direct injection, and an oversized TLB to provide “VMs as cheap as processes.”
ISA Independence
Hardware support for strongly ordered memory for high performance x86 emulation.
Unified Memory
Use a modern shared-memory architecture with high memory bandwidth for a stutter-free UX through no-copy memory sharing.
Linux Ready
Has supported drivers for every component of the SoC and keeps up with upstream kernel changes.
